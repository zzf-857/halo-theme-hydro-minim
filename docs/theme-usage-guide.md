# 氢·简 Hydro-Minim 使用教程

本文面向站长，按 Halo Console 中的主题配置分组说明 Hydro-Minim 的用法。配置入口通常在 `Halo Console -> 外观 -> 主题 -> 氢·简 -> 设置`。

上线前建议先完成三件事：配置站点基础信息、配置主菜单、安装你真正需要的插件。别上来就开一堆插件页，结果一个插件没装，前台全是安装提示，那场面多少有点“毛坯房摆家具”。

## 安装与启用

### 兼容版本

Hydro-Minim 要求 Halo `>=2.20.0`。

### 构建主题包

开发环境中运行：

```bash
pnpm install
pnpm run check
pnpm run build
```

构建成功后会生成：

```text
dist/halo-theme-hydro-minim-1.0.0.zip
```

在 Halo Console 上传并启用该主题包即可。

### 基础站点信息

建议先在 `Halo Console -> 系统 -> 站点设置` 中配置：

- 站点标题。
- 站点副标题。
- 站点 URL。
- 站点 Logo 或 favicon。
- SEO 描述。

主题的页脚、友链页和部分空状态会读取这些站点信息作为兜底。

## 品牌与站点

### 品牌标识

配置项：

- 顶部品牌展示：选择 Header 左侧展示 Logo 或站点名称。
- 方形标识：用于滚动后的胶囊菜单和移动端菜单。
- 深色方形标识：用于深色模式；留空时回退到浅色方形标识。
- 长方形 Logo：用于顶部菜单未收缩时的品牌区。
- 深色长方形 Logo：用于深色模式顶部菜单；留空时使用主题内置深色 Logo。
- Logo 替代文本：用于无障碍文本和图片 alt。

推荐做法：

- 方形标识使用 `1:1` 透明背景 SVG 或 PNG。
- 长方形 Logo 使用约 `3:1` 的横向图。
- 如果选择“展示站点名称”，Header 会直接显示 Halo 站点名称，不再依赖上传 Logo。

### 默认素材

配置项：

- 默认文章封面 1
- 默认文章封面 2
- 默认文章封面 3

当文章没有封面时，列表卡片会按顺序轮换使用这些默认封面。建议上传同一视觉系统下的三张图，比例尽量统一。当前仓库里默认图片文件是占位路径，上线前最好替换成真实图片，不然就别怪页面看起来像没装修完。

### 社交链接

配置项：

- 名称
- 链接
- 图标
- 新标签打开

社交链接用于首页关于模块和 Hydro 关于页联系区域。链接为空或为 `#` 时前台不会展示。图标使用 Iconify SVG，建议选择线性图标，保持主题细线风格。

常用示例：

```text
GitHub  https://github.com/yourname
Email   mailto:hello@example.com
RSS     /sitemap.xml
```

### 页脚信息

配置项：

- 显示页脚滚动字。
- 滚动字。
- 页脚品牌短字。
- 页脚描述。
- 显示页脚导航。
- 显示页脚站点统计。
- 版权文案。
- 构建署名。
- ICP 备案号与链接。
- 公安联网备案号与链接。

页脚站点统计读取 Halo 的 `siteStatsFinder.getStats()`，展示访问、文章、评论、点赞和分类数量。

版权文案支持占位符：

```text
{year} 当前年份
{site} 站点名称
```

示例：

```text
© {year} {site}. All rights reserved.
```

国内备案站点建议填写 ICP 和公安联网备案信息；非国内备案站点可以留空，前台不会显示。

## 视觉与动效

### 配色

配置项：

- 默认配色模式：跟随系统、浅色、深色。
- 浅色背景。
- 深色背景。
- 浅色强调色。
- 深色强调色。

访客没有手动切换前，会使用默认配色模式。访客点击 Header 深浅色按钮后，偏好会写入浏览器本地存储 `localStorage.hydro-color-scheme`。

建议：

- 浅色背景保持接近 `#F3F3F3` 的灰白纸面。
- 深色背景保持低饱和深灰，不建议纯黑。
- 强调色不要选高饱和霓虹色；主题本身是轻质感，不是夜店门头。

### 动效与质感

配置项：

- 显示轻微颗粒质感。
- 启用全局动效。
- 启用 Lenis 平滑滚动。
- 启用深浅色光幕过渡。
- 启用 Hero 图片跟随。
- 启用文章卡片 3D Hover。
- 启用图片灯箱交互。

说明：

- 关闭全局动效后，大部分 JS 动效、CSS 动画和过渡会停用。
- 图片灯箱不依赖全局动效，可单独控制。
- 卡片 3D Hover 主要作用于桌面端，触屏设备会自然降级。
- 主题尊重 `prefers-reduced-motion`，减少动效用户不会被强行动画糊脸。

### 深浅色切换文案

配置项：

- 切换到浅色提示。
- 切换到深色提示。

这些文案用于 Header 按钮的 `aria-label`、`title` 和移动端菜单动作文字。通常保留默认即可，除非你想统一站点语气。

## 导航与结构

### Header 与搜索

配置项：

- 显示搜索入口。
- 搜索入口方式：弹窗搜索或页面搜索。
- 允许访客切换深浅色。
- 移动端菜单按钮提示。

搜索入口依赖官方 `PluginSearchWidget`。如果插件未安装或未启用，主题不会展示无效搜索入口。

搜索入口方式：

- 弹窗搜索：适合希望用户不离开当前页面的站点。
- 页面搜索：适合希望有独立搜索页、可分享搜索链接的站点。

如果关闭“允许访客切换深浅色”，Header 和移动端菜单中的切换入口会隐藏，只使用后台默认配色模式。

### 默认导航

配置项：

- 无主菜单时的默认导航。

当 Halo 没有配置主菜单时，Header 和页脚会读取这里的链接。支持站内路径、锚点和完整 URL。

默认值：

```text
文章 -> #articles
分类 -> #categories
关于 -> #about
```

推荐上线时还是配置 Halo 主菜单。默认导航是兜底，不是长期偷懒方案。

### 分页

配置项：

- 上一页文案。
- 下一页文案。

用于列表页、插件页等分页组件。

### 浮动操作

配置项：

- 显示返回顶部按钮。
- 返回顶部显示阈值。
- 启用快捷操作按钮。
- 快捷操作提示。
- 全局默认快捷项。
- 页面快捷配置。

返回顶部阈值单位是像素。例如 `100` 表示向下滚动超过 100px 后显示返回顶部按钮。

#### 快捷项类型

跳转链接：

- 填写链接地址。
- 可选择是否新标签打开。

主题内置动作：

- 打开搜索。
- 切换深浅色。
- 返回顶部。
- 复制当前链接。
- 会员签到。
- 文章收藏。
- 打印页面。
- 打开移动菜单。
- 滚动到评论区。

自定义函数：

- 填写函数名。
- 填写 JSON 参数。

#### 页面快捷配置

匹配方式：

- 所有页面。
- 模板 ID。
- 永久链接。

匹配优先级：

1. 永久链接规则。
2. 模板 ID 规则。
3. 所有页面规则。
4. 全局默认快捷项。

常见模板 ID：

```text
post
page
archives
categories
tags
search
page_about
docs
photos
moments
links
```

永久链接示例：

```text
/about
/photos
https://example.com/about/
```

自定义函数示例：

```js
window.HydroMinimActions = {
  ...(window.HydroMinimActions || {}),
  openSponsorModal({ payload }) {
    const modal = document.querySelector(`[data-modal="${payload.modal}"]`);
    modal?.classList.add("is-open");
  },
};
```

快捷项中填写：

```text
函数名：openSponsorModal
参数：{"modal":"sponsor"}
```

## 首页

### 首屏 Hero

配置项：

- 显示首屏 Hero。
- Hero 标签。
- Hero 标题。
- Hero 描述。
- Hero 媒体类型：图片或视频。
- Hero 封面。
- Hero 视频。
- Hero 右侧文章数量。
- Hero 右侧指定文章。

媒体类型说明：

- 图片：直接展示 Hero 封面。
- 视频：先展示 Hero 封面作为 poster，视频加载成功后播放视频。

建议：

- Hero 标题保持短词或短句。
- Hero 描述控制在两行以内。
- 视频建议使用体积小、可循环播放、低干扰的短视频。
- 右侧指定文章可用于上线首屏推荐，不指定时自动使用首页最新文章。

### 最新文章

配置项：

- 显示最新文章。
- 最新文章标题。
- 首页文章展示数量。

展示数量范围为 1 到 6。内容站建议 3 到 4 篇，别把首页第一屏下面堆成“文章批发市场”。

### 分类模块

配置项：

- 显示分类模块。
- 分类模块标题。
- 首页分类展示数量。

展示数量范围为 1 到 8。适合把站点核心内容方向放在首页，帮助读者快速进入主题。

### 关于模块

配置项：

- 显示首页关于模块。
- 关于模块标签。
- 关于标题。
- 关于副标题。
- 关于段落一、二、三。
- 读者展示值。
- 显示社交链接。

读者展示值只是视觉文案，不会自动统计真实读者数。你可以填 `2.4k`、`120+`、`∞` 这种品牌化表达，但别写离谱数据，读者又不傻。

## 内容页面

### 文章详情

配置项：

- 显示文章封面 Hero。
- 显示作者信息。
- 启用赞赏区。
- 赞赏二维码。
- 显示上一篇/下一篇。
- 启用评论区。
- 启用文章目录。
- 启用阅读进度。
- 启用阅读操作栏。
- 启用点赞动作。
- 启用分享动作。
- 启用相关文章。
- 相关文章数量。

使用建议：

- 长文站点建议开启目录、阅读进度和阅读操作栏。
- 如果文章普遍很短，可以关闭目录。
- 开启赞赏区前必须上传清晰的 `1:1` 赞赏二维码。
- 评论区仍需要站点或评论插件支持，主题只负责渲染 Halo 评论组件。

移动端说明：

- 手机端会显示底部阅读条。
- 目录会以半屏抽屉打开。
- 上一篇/下一篇会变成适合手指点击的 pill 按钮。

### 独立页面

配置项：

- 显示页面封面。
- 显示发布时间。
- 启用评论区。
- 关于页显示统计。
- 关于页启用评论。

普通独立页使用 `page.html`，Hydro 关于页使用自定义模板 `page_about.html`。

创建 Hydro 关于页：

1. 进入 `Halo Console -> 内容 -> 页面`。
2. 新建页面，例如“关于”。
3. 页面模板选择 `Hydro 关于页`。
4. 填写正文、封面和发布状态。
5. 在菜单中添加该页面入口。

### 归档

配置项：

- 归档标题。
- 归档描述。
- 显示归档统计。
- 显示文章封面。

归档页按时间组织文章，适合读者回看历史内容。文章封面较多时建议开启封面；如果你更喜欢纯文本年表，可以关闭。

### 分类与标签

配置项：

- 分类页标题。
- 分类页描述。
- 分类页显示统计。
- 标签页标题。
- 标签页描述。
- 标签页显示统计。

分类页适合表达内容结构，标签页适合表达关键词网络。Hydro-Minim 的标签页使用抽象天空和云朵标签视觉，建议标签数量不要过少，否则天空会有点空；也别一口气几百个标签，那不是标签云，是关键词沙尘暴。

### 搜索

配置项：

- 搜索页标题。
- 搜索页描述。
- 搜索框占位。
- 搜索结果数量。
- 显示推荐文章。
- 推荐文章数量。

搜索页依赖搜索能力提供结果。推荐文章用于初始状态或无关键词时展示最近更新。

### 分享海报

配置项：

- 启用文章分享海报。
- 启用瞬间分享海报。
- 弹窗标题。

文章页和瞬间详情页可以显示分享海报入口。海报适合用户保存或二次传播。

## 插件页面

插件页面的共同原则：先安装对应插件，再创建页面或使用插件路由。主题会处理插件缺失状态，但站长别拿缺失状态当正式页面上线，太寒碜。

### 友链

配置项：

- 我的站点名称。
- 我的站点链接。
- 我的站点 Logo。
- 我的站点描述。
- 启用在线提交。
- 启用友链修改。
- 验证方式：邮箱验证码或图形验证码。
- 启用评论区。

使用步骤：

1. 安装并启用友链相关插件。
2. 如需在线提交，安装并启用 LinksSubmit 相关能力。
3. 创建独立页面，模板选择友链对应页面，或使用插件提供的友链路由。
4. 在主题设置中填写本站回链信息。
5. 根据站点审核流程决定是否开启在线提交和友链修改。

如果在线提交入口不显示，优先检查前台是否存在 `window.LinksSubmit`，别先怪主题。

### 瞬间

配置项：

- 每页数量。
- 启用点赞。
- 启用评论。
- 显示详情入口。

使用步骤：

1. 安装并启用 `PluginMoments` 或兼容瞬间插件。
2. 发布瞬间内容。
3. 需要作为独立页面时，新建页面并选择 `Hydro 瞬间页`。
4. 配置每页数量、点赞、评论和详情入口。

瞬间详情页支持分享海报。是否显示海报入口还会受“内容页面 -> 分享海报”配置影响。

### 图库

配置项：

- 每页数量。
- 启用评论区。

使用步骤：

1. 安装并启用图库插件。
2. 创建图库分组并上传图片。
3. 新建页面并选择 `Hydro 图库页`，或使用图库插件路由。
4. 根据图片量设置每页数量。

图库图片会使用主题灯箱交互。若关闭“视觉与动效 -> 图片灯箱交互”，点击放大能力会停用。

### 装备

配置项：

- 启用评论区。

使用步骤：

1. 安装并启用装备插件。
2. 创建装备分组和装备条目。
3. 新建页面并选择 `Hydro 装备页`。

装备页适合展示硬件、软件、摄影器材、开发工具等清单。

### 朋友圈

配置项：

- 每页数量。
- 启用评论区。

使用步骤：

1. 安装并启用朋友圈插件。
2. 添加订阅源并完成同步。
3. 新建页面并选择 `Hydro 朋友圈页`。
4. 根据订阅源活跃程度设置每页数量。

### 追番

配置项：

- 展示数量。
- 启用评论区。

使用步骤：

1. 安装并启用追番插件。
2. 添加追番条目。
3. 新建页面并选择 `Hydro 追番页`。
4. 设置最多展示数量。

### Steam

配置项：

- 最近游玩数量。
- 游戏库每页数量。
- 启用游戏外链。
- 启用评论区。

Steam 页面适合展示最近游玩、游戏库和兴趣偏好。高级设置中还有 `Steam 前端缓存分钟数`，用于减少频繁请求。

### Docsme 文档

配置项：

- 启用文档目录。
- 启用评论区。
- 插件安装链接。

使用步骤：

1. 安装 `plugin-docsme`，文档详情页建议版本 `1.4.0` 或更高。
2. 创建文档项目、目录和文档。
3. 需要入口页时，新建页面并选择 `Hydro 文档中心`。
4. 根据文档长度决定是否启用文档目录。

如果插件未安装或版本不兼容，主题会展示安装或升级提示。

## 高级设置

### 代码注入

配置项：

- Head HTML。
- Footer HTML。
- 自定义 CSS。
- 自定义 JavaScript。

使用场景：

- 添加站点验证 meta。
- 添加统计脚本。
- 微调主题样式。
- 注册 `window.HydroMinimActions` 自定义快捷操作。

注意：

- `自定义 CSS` 只填写 CSS 内容，不要包裹 `<style>`。
- `自定义 JavaScript` 只填写 JS 内容，不要包裹 `<script>`。
- 注入代码会影响全站，复制网上来路不明的脚本前先过脑子。

### HydroNotice 提示 API

主题内置统一提示：

```js
window.HydroNotice?.success("保存成功");
window.HydroNotice?.error("提交失败，请稍后再试", { title: "Error" });
```

完整示例：

```js
const notice = window.HydroNotice?.show({
  id: "links-submit-status",
  title: "Links",
  message: "友链申请已提交",
  variant: "success",
  duration: 4200,
  dismissible: true,
});

notice?.close();
```

建议给业务状态类提示传稳定 `id`，重复提示会替换旧提示，避免顶部刷成消息瀑布。

### 性能

配置项：

- Steam 前端缓存分钟数。

默认值为 `3` 分钟。调试 Steam 页面时可以设为 `0` 关闭缓存；正式站点不建议长期关闭。
