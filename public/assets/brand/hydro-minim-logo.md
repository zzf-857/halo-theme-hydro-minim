# 氢·简 / Hydro-Minim Logo

## Design Thought

This logo is built around three ideas:

- **Hydrogen as lightness**: the open circular orbit and small teal particle suggest the first element, motion, and airiness.
- **Minim as restraint**: the mark uses only a few rounded strokes, matching the theme's soft glass, pill navigation, generous whitespace, and monochrome editorial rhythm.
- **Series identity**: the central structure quietly reads as an abstract **H**, so it can represent both "Hydro" and the wider "氢" theme series without depending on a full Chinese character at small sizes.

## Assets

- `hydro-minim-mark.svg`: compact mark for nav, favicon candidates, avatars, and app icons.
- `hydro-minim-lockup.svg`: horizontal lockup for theme previews, README covers, and marketplace display.

## Usage Notes

- Use the mark at small sizes.
- Use the lockup when the name needs to be explicit.
- The mark inherits `currentColor`; keep it black or white in most UI contexts.
- Keep the teal particle as the only accent so the logo stays aligned with the theme's quiet visual system.
